# BlossomRPC
BlossomRPC，基于Netty，Nacos实现的RPC框架。

学习当前RPC框架能帮助你了解RPC的底层原理，并且了解RPC是如何快速整合Spring生态的。

同时，也能帮助你更好的了解代理模式。

学习当前项目能帮助你了解大厂的开发规范以及Netty的高级用法。从而帮助你深入了解Netty以及Nacos。
